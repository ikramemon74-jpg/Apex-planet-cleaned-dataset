# Apex Planet Internship – Task 1
## Data Cleaning and Exploratory Data Analysis

### Objective
Clean the sales dataset, improve data quality, and perform exploratory data analysis to identify business insights.

### Tools Used
- Excel
- SQL (SQLite)
- Power BI

### Dataset
Sales Dataset containing customer, product, category, city, quantity, unit price, and total sales information.

### Data Cleaning
- Removed duplicate records
- Handled missing values
- Corrected data types
- Standardized text values
- Checked data consistency

### Exploratory Data Analysis (EDA)
- Monthly Sales Trend
- Sales by Category
- Sales by Product
- Sales by Gender
- Sales by City
- Customer Age Distribution

### SQL Analysis
Business questions were answered using SQL queries, including:
- Top selling products
- Highest sales city
- Monthly sales trend
- Sales by category
- Customer distribution

### Dashboard
Power BI dashboard created to visualize KPIs and sales insights.

### Key Insights
- Electronics generated the highest sales.
- Sales recovered strongly in the final months.
- Male and female sales were nearly balanced.
- Customer distribution varied across cities.
